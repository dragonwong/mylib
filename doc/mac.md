# Mac 的快捷键

*约定：*
	
 \  | 1       | 2  | 3 | 4   |
:--:|:-------:|:--:|:-:|:---:|
 键 | command | ctr|alt|shift|
 符号| ⌘       | ^  | ⌥ | ⇧   |

## 系统

### 窗口

- 窗口全屏/还原 `^⌘F`

- 显示桌面 `F11`

- 切换全屏窗口 `ctr + ←→`
 
- mission control `ctr + ↑`

- 显示所有当前应用窗口 `ctr + ↓`

- 隐藏当前窗- 口 `⌘H`

- 隐藏除了当前窗口的所有其他- 窗口 `⌘⌥H`

- 最小化当前窗口 `⌘M`

### 其他

- 表情 & 字符 `ctr + ⌘ + space`

- 剪切 `⌘C + ⌘⌥V`

- hosts `/ect/hosts`

- finder -> terminal `将 finder 中的图标拖入 terminal 中`

### 截图

#### 系统

- 截屏并保存到桌面 `⌘⇧- 3`

- 截屏并保存到粘贴板 `⌘- ^⇧3`

- 截取指定区域并保存到桌面 `⌘⇧4`

- 截取指定区域并保存到粘贴板 `⌘^⇧4`

## chrome

## sublimetext 3

- 整行移动 `ctr + ⌘ + ↑↓`

- jsFormat `ctr + alt + f`

- 删除光标至行末 `⌘KK`
