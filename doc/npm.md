# npm

```
# npm path
/usr/local/lib/node_modules/

# 查看全局 npm 包
npm -g ls --depth=0

# 安装本地源码
cd <source-project>
npm install .

# 安装开发依赖，并写入 package.json 的 devDependencies 项
npm install gulp-util --save-dev

# npm 更新
npm install -g npm
```
